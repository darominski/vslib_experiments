  • cpha [rw]: Clock phase as defined in SPI spec.
  • cnv_pol [rw]: Polarity of CNV output. 0 - negative, 1 - positive
  • cnv_with_cs [rw]: CNV output is also CS output
  • busy_pol [rw]: Polarity of BUSY input. 0 - negative, 1 - positive

  • busy_time [rw]: Number of SPI clock cycles that controller should wait in BUSY state. Valid only if BUSY source is set to internal timer
  • clk_act_in_conv [rw]: SPI clock is active (1) or disabled (0) in conversion/BUSY state
  • data_width [rw]: ADC data width
